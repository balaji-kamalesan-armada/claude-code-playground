## Key Concepts You'll Learn

**CONTEXT** = What Claude sees RIGHT NOW
- Code, comments, files, conversation
- Temporary (only in current view)

**MEMORY** = What Claude remembers FOREVER
- Tech stack, architecture, conventions
- Persistent across all sessions
- Use `/remember` command

**SESSION CONTEXT** = Managing token budgets
- ~200k tokens per session
- Use `/compact` to save space
- Know when to start fresh

**SKILLS** = Custom workflows
- Create slash commands like `/commit`
- Automate repetitive tasks

**RULES** = Coding standards
- Define in `.claude/rules.md`
- Claude follows automatically
- Ensures consistency
