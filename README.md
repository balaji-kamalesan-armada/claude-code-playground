# Claude AI Playground

A comprehensive learning environment to explore Claude AI capabilities, from basic to advanced usage patterns.

**‚Üí New here? Read [HOW-TO-USE.md](HOW-TO-USE.md) first! ‚Üê**

## Journey: START ‚Üí BEST

This playground is designed to help you understand how to effectively work with Claude AI by comparing:
- **START-REPO**: Minimal setup with common pitfalls (demonstrates hallucination, confusion)
- **BEST-REPO**: Optimal setup with best practices (demonstrates clarity, accuracy)

## Learning Modules

### 1. [CONTEXT](01-CONTEXT/README.md) - 15 minutes
**What Claude sees RIGHT NOW**
- Learn what context is (code, comments, files, conversation)
- See examples of low vs high context
- Practice adding context to your code

### 2. [MEMORY](02-MEMORY/README.md) - 15 minutes
**What Claude remembers FOREVER**
- Learn the difference between memory and context
- Set up memory for your projects
- Practice the `/remember` command

### 3. [SESSION CONTEXT](03-SESSION-CONTEXT/README.md)
Master session context management
- Learn about context windows
- Understand token budgets
- Practice context exhaustion scenarios
- Strategies for efficient context usage

### 4. [SKILLS](04-SKILLS/README.md)
Create and use custom skills
- Build reusable workflows
- Create domain-specific helpers
- Integrate custom tools

### 5. [RULES](05-RULES/README.md)
Implement effective rules and guidelines
- **without-rules**: Observe inconsistent behavior
- **with-rules**: Establish clear boundaries and patterns

## Getting Started

1. **Start with START-REPO**: Experience common issues
2. **Work through each module**: Learn concepts progressively
3. **Study BEST-REPO**: See all practices combined
4. **Apply learnings**: Transform START into BEST

## Prerequisites
